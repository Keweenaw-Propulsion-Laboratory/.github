# Keweenaw Propulsion Labs (KPL) 🚀

Welcome to the official GitHub organization for **Keweenaw Propulsion Labs**. We are a student-led aerospace research and development group dedicated to advancing experimental propulsion systems, autonomous flight technologies, and hands-on aerospace engineering. 

Based in Michigan's Upper Peninsula, we push the boundaries of what's possible in rocketry and unmanned aerial systems (UAS).

---

## Current Projects

### [AFCDrone](https://github.com/Keweenaw-Propulsion-Laboratory/AFCDrone)
This project focuses on an automated flight system powered by electric propeller propulsion. The vehicle serves as a dynamic test platform to validate our custom control algorithms before they are integrated into a future liquid-fueled vehicle.

### [AFCDrone_BaseStation](https://github.com/Keweenaw-Propulsion-Laboratory/AFCDrone_BaseStation)
The ground control counterpart to the AFCDrone. This repository contains the telemetry and command software required to monitor and interact with the vehicle before, during, and after flight. 

### [simulink-testing](https://github.com/Keweenaw-Propulsion-Laboratory/simulink-testing)
The simulation and modeling hub for the AFCDrone flight algorithms. All vehicle dynamics and control loop simulations are built and tested using MATLAB & Simulink. 

### [AFC-Wiring-Diagram](https://github.com/Keweenaw-Propulsion-Laboratory/AFC-Wiring-Diagram)
The electrical architecture repository for the AFCDrone test vehicle, featuring our system wiring diagrams and schematics designed in KiCad.
